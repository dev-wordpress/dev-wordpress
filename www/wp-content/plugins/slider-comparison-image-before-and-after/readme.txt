Slider comparison image before and after for your website to WordPress.

== Description ==

The plugin Slider comparison image before and after allows you to create the effect of comparing the two images before and after. 

You can use the model vertically and horizontally. Used via shortcode [sciba].

== Installation ==

1. Unpack the archive with the plugin.
2. Download the folder with the plug-in directory `/wp-content/plugins/` on your server.
3. Activate the plugin Slider comparison image before and after in plugin section in the administration panel WordPress.
4. The panel will be the publication of a button insert shortcode sciba.
